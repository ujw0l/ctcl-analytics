CT Commerce Lite addon to display store analytics

== Description ==

Plugin to display Analytics of CT Commerce Lite ecommerce site where users and view sales report in chart


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/ctcl-analytics` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings-&gt;Plugin Name screen to configure the plugin
4. View analytics in Info tab Analytics subtab


== Frequently Asked Questions ==

= Where can I view analytics of store? =

You can it under Information tab in Analytics sub tab
